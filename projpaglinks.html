Página de Links

DETALHES DO PROJETO- Minha Página de Links Gabriela Vieira.

Este projeto é uma página de links desenvolvida para reunir e compartilhar, de forma organizada, acessos rápidos a plataformas, redes sociais, portfólio e outros conteúdos importantes. O objetivo é criar uma interface simples e responsiva, que se adapte a diferentes dispositivos e ofereça fácil navegação ao usuário

Tecnologias Utilizadas:

  • HTML
  • CSS

Meu Papel no Projeto:

Fui responsável por todo o desenvolvimento deste projeto, desde o planejamento até a implementação final. Todo o trabalho foi focado em oferecer uma navegação simples, eficiente e organizada para o usuário.
